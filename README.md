# ⚡ ChargeEasy

ChargeEasy is a modern web application that helps users locate and navigate to nearby electric vehicle (EV) charging stations using an interactive map interface. It provides real-time location insights and smooth navigation for EV users.

---

## 📌 Features

* 🔌 Discover nearby EV charging stations
* 🗺️ Interactive map using Leaflet
* 📍 Location-based station search
* 🧭 Route navigation with directions
* ⚡ Fast and responsive UI (React + Vite)
* 🎨 Styled with Tailwind CSS
* 🗂️ State management using Zustand

---

## 🛠️ Tech Stack

* **Frontend:** React, Vite
* **Styling:** Tailwind CSS
* **Maps & Routing:** Leaflet, React-Leaflet
* **State Management:** Zustand

---

## ⚙️ Installation & Setup

Follow these steps to run the project locally:

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/chargeeasy.git
cd chargeeasy
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run the development server

```bash
npm run dev
```

### 4️⃣ Open in browser

Visit:

```
http://localhost:5173
```
